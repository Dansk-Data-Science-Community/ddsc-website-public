{% translate "Behandlingsgrundlag og formål" %}

{% blocktrans trimmed %}Dine almindelige kontaktoplysninger som navn indhenter vi for at kunne adressere dig som person. Din e-mailadresse indhenter vi for at kunne sende dig nyhedsbreve og for at give dig en sikker loginmetode{% endblocktrans %}

Du kan til enhver tid til- og afmelde dig nyhedsbrevet på din brugers profil. Du kan til enhver tid tilbagekalde dit samtykket og stoppe behandlingen af dine persondata.{% endblocktrans %}

Fx tidligere tilmeldte events.{% endblocktrans %}

{% translate "Andre modtagere af personoplysninger" %}

{% blocktrans trimmed %}Vi sælger ikke dine persondata til tredjepart, og vi overfører ikke dine persondata til tredjelande.{% endblocktrans %}

{% blocktrans trimmed %}Vi har vores hjemmeside hos DigitalOcean, som fungerer som vores databehandler. Alle persondata som du oplyser på vores hjemmeside vil blive opbevaret i DigitalOceans datacentre indenfor EU. Databehandleraftalen imellem DigitalOcean og Dansk Data Science Community kan læses i deres{% endblocktrans %} {% translate "generelle databehandleraftale" %}.

{% blocktrans trimmed %}Vi anvender ikke eksterne virksomheder som leverandører af services eller andet, der gør brug af dine persondata til kommercielle formål.{% endblocktrans %}
